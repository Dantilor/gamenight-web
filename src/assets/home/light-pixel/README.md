# GNH Light Pixel Perfect Pack

Это не новые сгенерированные картинки. Это PNG-элементы, вырезанные из утвержденного экрана.

## Куда положить в проект

Папку `src_assets_home_light_pixel` переименуй/переложи в:

`src/assets/home/light-pixel/`

Итог:
`src/assets/home/light-pixel/light_hero_exact.png`
и т.д.

## Как добиться точь-в-точь

Cursor НЕ должен перерисовывать портал, иконки, карточки и кнопки.
Он должен использовать эти PNG как визуальные слои/фон карточек и поверх них накладывать прозрачные кликабельные зоны.

Это самый надежный способ получить точный вид.
Потом можно постепенно переверстать в CSS, но сначала делаем pixel-perfect.
